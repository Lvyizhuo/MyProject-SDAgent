### 现象：查询 "MacBook 价格" 未调用 webSearch 工具

**问题描述**：用户输入"查一下MacBook的价格"，系统未调用 webSearch 工具，而是返回了 RAG 检索到的模板化回答。

**可能原因分析**：

| 原因 | 说明 | 验证方式 |
|------|------|----------|
| 1. RAG 命中 | 知识库中存在"关于产品价格"的预定义文档，LLM 直接使用检索结果 | 查看 LoggingAdvisor 中的 [RAG] 日志 |
| 2. isPolicyPriceMixedQuestion 干扰 | 问题同时包含"政策"和"价格"关键词，导致 enforceRealtimeWebSearch 被跳过 | 查看 ChatService.java 第 474 行逻辑 |
| 3. ReAct 规划未生成 webSearch | 大模型在规划阶段判定"不需要联网" | 查看 planningService 的日志输出 |
| 4. ToolIntentClassifier 误拦截 | 虽然 hasExplicitModelInfo 包含 "macbook"，但仍可能被其他逻辑拦截 | 查看 ToolIntentClassifier 日志 |

# 需求1：新增第一阶段 FastPath 高频直答规则

**需求说明**：在现有 `FastPathService` 中，新增 **P0/P1/P2 优先级**的快速应答规则，覆盖问候、能力咨询、补贴比例、申请材料、办理流程、到账时间、地区补贴查询等高频问题，实现不经过 LLM 与工具调用的直接回复，提升响应效率。

**设计原则**：
- ✅ 只将**纯静态知识**放入 FastPath
- ❌ 不要放入需要实时数据的问题（如"今天的价格"）
- ❌ 不要放入需要复杂计算的问题（如"算补贴"）
- ⚠️ 补贴比例等可变内容使用**通用描述**，避免硬编码具体数值

**代码示例**：

```java
package com.shandong.policyagent.service;

import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Service;

import java.util.Locale;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

@Slf4j
@Service
@RequiredArgsConstructor
public class FastPathService {

    private static final Pattern CITY_PATTERN = Pattern.compile("([\\p{IsHan}]{2,8}市)");

    // 动态配置注入：避免硬编码可变政策内容
    private final DynamicAgentConfigHolder dynamicAgentConfigHolder;

    public String tryDirectAnswer(String rawQuestion, SessionFactCacheService.SessionFacts facts) {
        if (rawQuestion == null || rawQuestion.isBlank()) {
            return null;
        }

        String normalized = normalize(rawQuestion);

        // ========== P0: 极高频（80% 请求） ==========

        // 规则 1: 问候语
        if (isGreeting(normalized)) {
            log.debug("FastPath 命中: 问候语");
            return "您好，我是山东省以旧换新政策咨询助手。您可以直接告诉我商品类型、购买价格和所在地市，我可以帮您估算补贴并说明申领流程。";
        }

        // 规则 2: 能力询问
        if (isCapabilityQuestion(normalized)) {
            log.debug("FastPath 命中: 能力询问");
            return "我可以帮您做三类事情：\n"
                    + "1. 解释山东省及各地市以旧换新补贴政策\n"
                    + "2. 根据商品类型和价格估算补贴金额\n"
                    + "3. 提供申请条件、所需材料和办理步骤\n\n"
                    + "如果您愿意，现在就可以发我：商品类型 + 购买价格 + 所在地市。";
        }

        // ========== P1: 高频（15% 请求） ==========

        // 规则 3: 补贴比例问题（使用动态配置，避免硬编码）
        if (isSubsidyRateQuestion(normalized)) {
            log.debug("FastPath 命中: 补贴比例");
            return dynamicAgentConfigHolder.getSubsidyRateIntro();
        }

        // 规则 4: 申请材料
        if (isMaterialQuestion(normalized)) {
            log.debug("FastPath 命中: 申请材料");
            return "常见申领材料包括：身份证明、购买发票/订单、支付凭证、旧机回收或交售凭证（按活动要求）、收款账户信息。\n"
                    + "不同地市活动会有细节差异，我可以按您所在城市给出更精确的材料清单。";
        }

        // 规则 5: 申请流程
        if (isApplicationProcessQuestion(normalized)) {
            log.debug("FastPath 命中: 申请流程");
            return "一般流程是：确认资格和品类 -> 在活动渠道下单并完成支付 -> 提交发票与回收等材料 -> 审核通过后发放补贴。\n"
                    + "把您的城市和商品类型告诉我，我可以给您对应渠道和操作要点。";
        }

        // 规则 6: 到账时间
        if (isArrivalTimeQuestion(normalized)) {
            log.debug("FastPath 命中: 到账时间");
            return "补贴到账时间通常取决于活动平台与审核进度，常见为审核通过后几个工作日到数周内。\n"
                    + "如果您告诉我申领渠道（平台/门店）和城市，我可以给您更具体的时效参考与催办建议。";
        }

        // ========== P2: 中频（需要 SessionFact） ==========

        // 规则 7: 特定地区是否有补贴
        if (isLocalSubsidyQuestion(normalized)) {
            String city = resolveCityHint(facts, rawQuestion);
            if (city != null && !city.isBlank()) {
                log.debug("FastPath 命中: 本地补贴查询 | city={}", city);
                return city + "通常会结合省级政策开展本地补贴活动，但具体品类、名额和时间以当地最新公告为准。\n"
                        + "您可以继续告诉我商品类型和预算，我会按该城市口径帮您判断是否适用。";
            }
        }

        return null;
    }

    // ========== 匹配判断方法（按优先级排序） ==========

    private boolean isGreeting(String normalized) {
        return normalized.matches("^(你好|您好|嗨|hello|hi|在吗|有人吗)[!！。,.?？ ]*$");
    }

    private boolean isCapabilityQuestion(String normalized) {
        return containsAny(normalized, "你能做什么", "你会什么", "怎么用", "可以帮我", "功能");
    }

    private boolean isSubsidyRateQuestion(String normalized) {
        return containsAny(normalized, "补贴比例", "补贴几个点", "按多少补贴", "国补比例");
    }

    private boolean isMaterialQuestion(String normalized) {
        return containsAny(normalized, "申请材料", "需要什么材料", "要准备什么", "提交什么资料", "要哪些资料");
    }

    private boolean isApplicationProcessQuestion(String normalized) {
        return containsAny(normalized, "申请流程", "怎么办理", "怎么申请", "怎么操作", "申领流程", "领取流程");
    }

    private boolean isArrivalTimeQuestion(String normalized) {
        return containsAny(normalized, "多久到账", "多久到", "多久能到账", "什么时候到账", "发放时间", "补贴到账");
    }

    private boolean isLocalSubsidyQuestion(String normalized) {
        return containsAny(normalized,
                "本地有补贴吗", "当地有补贴吗", "我们这有补贴吗", "这个城市有补贴吗", "有地方补贴吗", "市补贴");
    }

    // ========== 辅助方法 ==========

    private String resolveCityHint(SessionFactCacheService.SessionFacts facts, String rawQuestion) {
        if (facts != null && facts.getRegions() != null && !facts.getRegions().isEmpty()) {
            String region = facts.getRegions().iterator().next();
            if (region != null && region.endsWith("市")) {
                return region;
            }
        }
        if (rawQuestion != null && !rawQuestion.isBlank()) {
            Matcher matcher = CITY_PATTERN.matcher(rawQuestion);
            if (matcher.find()) {
                return matcher.group(1);
            }
        }
        return null;
    }

    private boolean containsAny(String text, String... keywords) {
        if (text == null || text.isBlank()) {
            return false;
        }
        for (String keyword : keywords) {
            if (text.contains(keyword.toLowerCase(Locale.ROOT))) {
                return true;
            }
        }
        return false;
    }

    private String normalize(String text) {
        return text == null ? "" : text.trim().toLowerCase(Locale.ROOT);
    }
}
```

# 需求2：价格查询异步预取中间层（非阻塞）

**需求说明**：在"澄清提问"之后、"Agent 规划"之前，新增一层**异步价格预取**。对于明确的价格查询意图，提前调用 webSearch 并缓存结果，但不阻塞主链路。主链路执行到 Agent 阶段时，优先检查缓存是否已命中。

**设计原则**：
- ✅ **异步执行**：不阻塞 FastPath → QA Cache → Agent 主链路
- ✅ **结果缓存**：存入 `ProductPriceCacheService`，供 Agent 阶段读取
- ✅ **失败静默**：预取失败不影响主流程，仅记录日志
- ✅ **条件触发**：仅对"纯价格查询"触发预取

**触发条件**：
```java
// 仅在以下情况触发预取：
// 1. 包含"价格/报价/多少钱"等关键词
// 2. 不包含"政策/补贴/以旧换新"等混合意图关键词
// 3. 有明确的品牌/型号信息（避免无效搜索）
```

```java
/**
 * 判断是否为"纯价格查询"（不含政策混合意图）
 */
private boolean isPurePriceQuery(String rawQuestion, SessionFactCacheService.SessionFacts facts) {
    if (rawQuestion == null || rawQuestion.isBlank()) {
        return false;
    }
    String normalized = rawQuestion.toLowerCase(Locale.ROOT);

    // 必须包含价格相关词
    boolean hasPriceIntent = containsAny(normalized,
            "价格", "报价", "多少钱", "市场价", "售价", "价位", "优惠");
    if (!hasPriceIntent) {
        return false;
    }

    // 不能包含政策/补贴混合意图
    boolean hasPolicyIntent = containsAny(normalized,
            "政策", "补贴", "国补", "以旧换新", "申领", "流程", "资格", "条件");
    if (hasPolicyIntent) {
        return false;
    }

    // 最好有明确的品牌/型号（避免无效搜索）
    return hasExplicitBrandOrModel(normalized, facts);
}

private boolean hasExplicitBrandOrModel(String normalized, SessionFactCacheService.SessionFacts facts) {
    // 检查 SessionFact 中是否已有商品信息
    if (facts != null && facts.getDeviceModels() != null && !facts.getDeviceModels().isEmpty()) {
        return true;
    }
    // 检查当前查询是否包含品牌/型号关键词
    return containsAny(normalized,
            "iphone", "ipad", "macbook", "thinkpad", "surface", "matebook",
            "华为", "小米", "荣耀", "oppo", "vivo", "三星", "联想", "戴尔", "惠普");
}
```

## 3.4 优化 ToolIntentClassifier（商品型号识别与意图控制）

**代码示例**：

```java
package com.shandong.policyagent.agent;

import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;

import java.util.Locale;
import java.util.Set;

@Slf4j
@Component
public class ToolIntentClassifier {

    // ========== 改进版：webSearch 分类逻辑 ==========

    private IntentDecision classifyWebSearch(String normalized) {
        boolean productOrPriceIntent = containsAny(normalized,
                "价格", "报价", "多少钱", "优惠", "参数", "配置", "型号", "款", "对比", "电商", "官网");

        // 非商品价格类实时查询 → 允许
        if (!productOrPriceIntent && containsAny(normalized, "最新", "实时", "新闻", "政策", "动态")) {
            return IntentDecision.allow("webSearch", "非商品价格类实时查询，允许联网检索");
        }

        // 商品价格类查询 → 检查是否有明确型号
        if (hasExplicitModelInfo(normalized)) {
            return IntentDecision.allow("webSearch", "已识别明确品牌/型号/规格，允许联网查询");
        }

        // 只有品牌但无型号 → 给更友好的提示
        if (hasBrandOnlyNoModel(normalized)) {
            return IntentDecision.block(
                    "webSearch",
                    "为避免查错，请补充具体型号（例如：MacBook Pro 14寸 M3、华为 Mate 70 Pro）。",
                    "webSearch 缺少具体型号信息"
            );
        }

        // 完全没有品牌/型号 → 拦截
        return IntentDecision.block(
                "webSearch",
                "你想查询哪个具体商品的价格？请提供品牌和型号，例如：iPhone 15 Pro Max 256GB。",
                "webSearch 缺少明确型号信息"
        );
    }

    // ========== 改进版：型号识别逻辑 ==========

    private boolean hasExplicitModelInfo(String normalized) {
        // 1. 品牌 + 数字/系列组合
        if (normalized.matches(".*(iphone|ipad|macbook|thinkpad|surface|matebook)\\s+[a-z0-9+\\-]+.*")) {
            return true;
        }
        // 2. 中文品牌 + 系列
        if (containsAny(normalized, "华为", "小米", "荣耀", "oppo", "vivo")
                && containsAny(normalized, "mate", "pro", "max", "ultra", "air", "mini")) {
            return true;
        }
        // 3. 存储规格（GB/TB）
        if (normalized.matches(".*\\d{2,4}(gb|g|tb|t).*")) {
            return true;
        }
        // 4. 年份款
        if (normalized.matches(".*202[3-6]款.*")) {
            return true;
        }
        return false;
    }

    private boolean hasBrandOnlyNoModel(String normalized) {
        boolean hasBrand = containsAny(normalized,
                "iphone", "ipad", "macbook", "thinkpad", "surface", "matebook",
                "华为", "小米", "荣耀", "oppo", "vivo", "三星", "联想", "戴尔", "惠普");
```
